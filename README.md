## How to run this repo in local dev

```
$ cp .env.example .env
```

Edit .env values and put the correct values for API keys and other configurations.

```
$ yarn
$ yarn dev
```

## How to run tests
```
$ yarn test
$ yarn test:unit
$ yarn test:e2e
```
